Snake-Game

Rules:

  • Control the snake eat apples that appear randomly on the board, and avoid colliding with the walls of the board or with the snake's own body.
  • The player controls the snake using the 4 arrow keys on the keyboard. The snake moves at a constant speed.
  • The player can reset the game at any time by refreshing the page. The game is won when the snake has eaten all the apples on the board.
